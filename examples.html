Growing line spinner

It should work with all Angular 1.x versions

This loader should offer the option to change the stroke width and size as required and the colour as needed along with the opacity. There is also a line version.

Circular

Most basic options - these must be set: give it an id and a type and youre good to go
<gl-spinner gl-id="loader0" gl-type="circle"></gl-spinner>

This simply loads in the default values for a very basic spinner.

Change the size. You might need it to fill a larger space - or simply a space in a line where a small amount of dynamic data is being loaded. Remember that it is positioned absolutely and takes the dimensions of the containing element that must be positioned relatively to sit in the right place. A tweak to the 'g' element in the css file using translate: transform() might be necessary.
<gl-spinner gl-id="loader1" gl-type="circle" gl-size="76"></gl-spinner>

Change the stroke thickness. There is a balance here, but the smaller the size, the thinner the line. I could have added that as a rule, but decided to leave that option to the end user.
<gl-spinner gl-id="loader2" gl-type="circle" gl-size="76" gl-stroke-width="1"></gl-spinner>

Change the colour. Any valid css colour will do and is applied to the element's 'stroke' property, so I thought I'd keep it that way.
<gl-spinner gl-id="loader3" gl-type="circle" gl-size="76" gl-stroke="#ed1b1e"></gl-spinner>

Change the opacity. Accepts the css values for opacity from 0 to 1 with all of the bits in between.
<gl-spinner gl-id="loader4" gl-type="circle" gl-size="76" gl-stroke="#ed1b1e"></gl-spinner>

Change the speed. If the spinner is small, it will be super fast, so with a size of 18, a speed of .25 isn't too bad. Large ones can be slow, so have a play.
<gl-spinner gl-id="loader5" gl-type="circle" gl-size="76" gl-stroke="#ed1b1e" gl-speed="5"></gl-spinner>

Don't forget to add the styles above for the line version ( glspinner[cc-type='line'] {width:100%;display:block;} )
<gl-spinner gl-id="loader6" gl-type="line" gl-size="76" gl-stroke-width="4" gl-stroke="#bada55" gl-opacity=".8" gl-speed="5"></gl-spinner>

The only attribute that does not affect the line version is the size.
